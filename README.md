## Tasks Manager

Task Manager is a task management system.
It is a web application that helps to manage users' tasks. Here you can add, update and delete tasks, statuses of tasks, and labels that can be added to tasks.

## Requirements

* JDK 20
* Gradle 8.4
* GNU Make
